# CQL Trace Viewer

## Usage

### Install Requirements

```
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt 
```

### Start App

```
python app.py
```

### Deployment

```
gcloud app deploy
```

## Features

- [x] Display table
- [x] Display scatter plot
- [x] Display message graph
- [ ] Select CQL / C* / DSE version
