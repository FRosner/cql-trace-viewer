# CQL Trace Viewer

## Usage

### Install Requirements

```
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt 
```

### Start App

```
python app.py
```

## Features

- [x] Display table
- [ ] Display message graph
- [ ] Display gantt chart
